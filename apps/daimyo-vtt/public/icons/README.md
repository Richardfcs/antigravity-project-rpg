# PWA Icons

Coloque os seguintes arquivos nesta pasta:

| Arquivo | Tamanho | Uso |
|---|---|---|
| `icon-192x192.png` | 192×192 px | Ícone padrão (home screen, app switcher) |
| `icon-512x512.png` | 512×512 px | Splash screen, instalação |
| `icon-maskable-192x192.png` | 192×192 px | Android adaptive icon (safe zone interna) |
| `icon-maskable-512x512.png` | 512×512 px | Android adaptive icon grande |
| `apple-touch-icon.png` | 180×180 px | iOS home screen (mova para `/public/`) |

## Dica para Maskable Icons

Use https://maskable.app/editor para garantir que o ícone tenha padding adequado na safe zone.
